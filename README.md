## 🚀 Cómo empezar
1. Clona este repositorio:

2. Abre la carpeta en tu editor de código (VS Code recomendado).
3. Para visualizar el proyecto en local, utiliza la extensión **"Live Server"** de VS Code para evitar problemas de CORS al manejar el LocalStorage.
